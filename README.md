# Financial data manipulation - Project Overview

This project demonstrates practical analytical tasks performed using Google Sheets / Excel, including data manipulation, reporting automation, pivot tables, and trend analysis.

Key Tasks Completed
1. Google Sheets Data Manipulation
Identified and highlighted duplicate trader IDs while considering different reporting weeks within the dataset.
Built a dynamic query in the GQuery sheet to return: Trader IDs, City, Reporting Week, Total Transactions, Total Gross Bookings and Currency for traders with more than 50 transactions. Applied formulas in the Formulas sheet to automatically calculate country-level metrics for the latest reporting week based on the raw dataset.
2. Pivot Table & Reporting
Created a pivot table for cities in Poland presenting: Number of traders, Total gross transactions, Average gross transactions and Completed transactions. The report was formatted for readability and business reporting purposes.
3. Sales & Transactions Analysis
Performed trend analysis for the Polish market by comparing:, Overall sales and Number of transactions across all reporting weeks. Additionally, created a visualization/dashboard chart to present performance trends and highlight key business insights over time.
